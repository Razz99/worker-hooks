# AGENTS.md

You are an expert in JavaScript, Rspack, Rsbuild, and library development. You write maintainable, performant, and accessible code.

## Commands

- `pnpm run build` - Build the library for production
- `pnpm run dev` - Turn on watch mode, watch for changes and rebuild the library
- `pnpm run test` - Run tests
- `pnpm run test:watch` - Run tests in watch mode

## Tools

### Biome

- Run `pnpm run lint` to lint your code
- Run `pnpm run format` to format your code
